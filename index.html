APS

   O projeto visa o desenvolvimento de um sistema de controle de reserva para cinema, abrangendo tanto os clientes quanto os colaboradores. O sistema permitirá a reserva de assentos para sessões, bem como a reserva de salas e datas para eventos este trabalho visa apresentar uma empresa que adota medidas de sustentabilidade de acordo com a ONU, através do desenvolvimento de um web site (utilizando HTML, CSS e Javascript).
